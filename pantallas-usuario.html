Perfil de Usuario
Pantalla para configurar la información personal del usuario, que se utilizará para personalizar las recomendaciones nutricionales y calcular las necesidades calóricas diarias.

Funcionalidades

  • Registro de datos personales (nombre, edad, peso, altura)
  • Selección de objetivos (mantenimiento, pérdida de peso, ganancia muscular)
  • Cálculo automático de IMC y necesidades calóricas
  • Almacenamiento seguro de datos de perfil
  • Sincronización con preferencias dietéticas

Preferencias Dietéticas
Configuración de restricciones alimentarias, alergias, intolerancias y preferencias dietéticas para personalizar las recomendaciones de recetas y evitar ingredientes problemáticos.

Funcionalidades

  • Selección de tipo de dieta (vegetariano, vegano, keto, etc.)
  • Configuración de alergias e intolerancias alimentarias
  • Preferencias nutricionales (bajo en sodio, alto en proteína, etc.)
  • Visualización de resumen de preferencias
  • Filtrado automático de recetas según restricciones

Escaneo OCR
Permite a los usuarios escanear ingredientes usando la cámara del dispositivo. Utiliza tecnología de reconocimiento óptico de caracteres (OCR) para identificar automáticamente los ingredientes y agregarlos a la búsqueda.

Funcionalidades

  • Activación de cámara para escanear ingredientes
  • Reconocimiento automático de texto en etiquetas de productos
  • Visualización de ingredientes detectados como chips
  • Posibilidad de eliminar ingredientes incorrectos
  • Integración con la base de datos de ingredientes de la app
